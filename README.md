# SF4 Addons

## 🚀 About
**SF4 Addons** is a custom addon collection for **Slimefun 4.9-UNOFFICIAL** on **Paper / Purpur 1.21.11**.

This repository is focused on:
- 🔧 updating older Slimefun addons for modern server versions
- 🧩 keeping addons compatible with the updated SF4 core
- 📦 publishing clean builds for owned-server use and testing

## 💻 Compatibility
- **Minecraft:** `1.21.11`
- **Server Software:** `Paper / Purpur`
- **Core:** `Slimefun 4.9-UNOFFICIAL`
- **Java:** `21`
